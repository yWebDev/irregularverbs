# Iverbs

Angular client for irregular verbs (Angular 21). For **CLI commands, build targets, assets, and SonarCloud setup**, see [docs/ANGULAR_CLI.md](docs/ANGULAR_CLI.md).

## Quick start

- `npm run start:dev` — local dev server with default environment and proxy.
- `npm start` — staging configuration and staging proxy.
- `npm run build` / `npm run build:prod` — application build (`dist/browser`).
- `npm test` — Karma + Jasmine.
- `npm run e2e` — Playwright.

Official reference: [Angular CLI documentation](https://angular.dev/tools/cli).
